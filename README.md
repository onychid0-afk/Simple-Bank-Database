# Simple-Bank-Database
A simple school project made in python to simulate a database.


**Purpose**

This was a 1 month project was made as a remix to another activity I did in class where we made a banking system but it didn't involve a database. Another project, which was a mars rover, required excel to make a database of all the scanned points a rover moves to. So in this project I mashed the two together, a banking system and a database.
